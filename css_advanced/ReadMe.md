HTML & CSS — Homepage Styling Exercise

This project is designed as a hands-on exercise to strengthen your CSS and layout skills by building and styling a homepage for a class assignment. It emphasizes modern CSS techniques, responsive design, and maintainable styling practices.

Project Overview

Goal: Style a homepage to practice core CSS concepts including selectors, the box model, Flexbox/Grid layouts, typography, colors, and responsive breakpoints.

Focus: Developing a visually appealing and functional homepage while improving CSS workflow and organization.

Files and Structure

[html_advanced/index.html] — The homepage to style.

[html_advanced/base.html] — Reference base template.

[html_advanced/README.md] — Project overview and instructions.

[html_advanced/images] — Image assets for use in the project.

[css_basic/styles.css] — Primary stylesheet to edit or extend. You may also create a separate CSS file and link it to index.html.

How to Preview Locally

Open html_advanced/index.html in your web browser, or use your editor's live preview feature.

Make CSS changes in css_basic/styles.css (or your custom stylesheet).

Refresh the browser to see your updates immediately.

Learning Objectives

Apply CSS selectors effectively and keep styles organized and maintainable.

Build layouts using Flexbox and Grid.

Implement responsive design for mobile and desktop devices.

Enhance visual hierarchy using typography, spacing, and color.
